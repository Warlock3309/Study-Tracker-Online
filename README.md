## Project Goal

This project aims to:  
- Allow users to log their daily study and personal activities.  
- Generate **daily scores** that reflect performance.  
- Enable progress comparison with friends.  
- Encourage consistent habits through gamification.

- ## Planned Features

- Daily activity and task logging  
- Automatic scoring system based on completed goals  
- Dashboard with progress charts and statistics  
- Score comparison between users (ranking system)  
- Option to add notes or comments for each activity  
- Responsive and user-friendly interface  
- Notifications or reminders to maintain daily routines

- ## Technologies

- **HTML / CSS / JavaScript(maybe)** (front-end)
